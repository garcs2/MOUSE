# Examples

These examples help to learn using MOUSE:

## Examples to calculate bottom-up cost estimate for some reactor types
* `watts_exec_LTMR.py`: Cost estimte for liquid metal microreactor
* `watts_exec_GCMR_Design_A.py' Cost estimte for gas-cooled microreactor
* `watts_exec_GCMR_Design_A.py' Cost estimte for gas-cooled microreactor
* `watts_exec_GCMR_Design_B.py' Cost estimte for another design of a gas-cooled microreactor
* `watts_exec_HPMR.py' Cost estimte for heat-pipe microreactor

## Examples for parametric studies
* `watts_exec_LTMR_UO2_vs_TRIGA.py` : The impact of changing the fuel (UO2 vs UzrH) on the cost
* `watts_GCMR_packing_fraction.py` : The impact of changing the TRISO particles packing fraction on the cost
